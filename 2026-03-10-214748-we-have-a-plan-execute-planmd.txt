● Clean. Here's what was delivered for this phase:

  Profiling infrastructure:
  - GpuProfiler — rolling N-frame history of per-pass GPU timings with averages
  and a formatted summary (profiler.rs)
  - TimestampQueries — wraps wgpu::QuerySet to insert begin/end timestamps
  around each compute pass (timestamp.rs)

  Shader optimisations:
  - basic_adjust.wgsl — fused tone+color in a single compute pass; when these
  two ops appear consecutively in a layer, the renderer now routes them through
  BasicAdjustPipeline, saving one full texture read/write round-trip
  - sharpen_h.wgsl + sharpen_v.wgsl — replaced the old 3×3 box blur with a
  proper 7-tap separable Gaussian USM; better edge quality at the same cost, and
   more cache-friendly memory access

  CPU-side optimisations:
  - TextureCache — source images are cached on the GPU by a u64 key; repeated
  renders no longer re-upload unchanged pixel data

  What would you like to tackle next? Remaining Phase 5 items are: undo/redo,
  project save/load (.shade ZIP), keyboard shortcuts/pan/zoom, ICC colour
  management, and CI test setup.

  shade-core — Three new types:
  - ColorSpace enum: Srgb, LinearSrgb, AdobeRgb, DisplayP3, ProPhotoRgb, Custom,
   Unknown
  - ColorMatrix3x3 — 3×3 transform with pre-computed const matrices for Adobe
  RGB, Display P3, ProPhoto RGB (and their inverses)
  - ProjectColorSettings — working / display / export colour space triplet

  shade-io — Full ICC pipeline:
  - load_image_with_colorspace() — reads JPEG APP2 segments and PNG iCCP/sRGB
  chunks to detect the embedded profile; identifies Adobe RGB, Display P3,
  ProPhoto RGB, sRGB by name or profile data
  - to_linear_srgb() — in-place CPU conversion to the internal working space
  using the proper sRGB piecewise EOTF and colour matrices
  - from_linear_srgb() — inverse for display/export encoding

  shade-gpu — color_transform.wgsl + ColorTransformPipeline:
  - 6 GPU modes: identity, linear→sRGB, sRGB→linear, matrix-only,
  gamma-decode+matrix, matrix+sRGB-encode
  - ColorTransformUniform::to_linear_srgb() / from_linear_srgb() factory methods
   build the correct uniform from a ColorSpace automatically
  - Renderer::apply_color_transform() for GPU-side display transform at viewport
   output

  shade-cli — now auto-detects source colour space from embedded profiles, with
  --color-space and --display-space overrides if needed.
